## 项目简介

现在同人音乐的信息比较分散，各个平台的标签和分类也不太统一。比如：
- VocaDb与TouhouDb的数据不互通，只依靠外链
- 其标签与分类不够细致，准确
- 用户界面不够友好，现代

我们想做一个更好用的数据库平台，解决这些问题

**A**：
- 更快更稳：使用最新技术栈，从服务器到界面会精心优化
